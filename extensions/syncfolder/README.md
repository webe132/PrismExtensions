# SyncFolder

Keeps a chosen folder in sync across all of your instances. A file saved while
playing one version becomes available in every other instance. The newest copy
of each file always wins.

## Settings

| Setting | Description |
|---------|-------------|
| Folder to sync | Which folder to keep in sync: **Shaders** (`shaderpacks`), **Resource packs** (`resourcepacks`), **Schematics** (`schematics`) or **Worlds** (`saves`). |
| Show a notification after syncing | Pop up a message when files are copied. |

Open the settings from the **Extensions Settings** page: select this extension and
click the gear **Settings** button.

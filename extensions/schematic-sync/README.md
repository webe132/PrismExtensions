# Schematic Sync

Keeps the **schematics** folder in sync across all of your instances. A schematic
saved while playing one version becomes available in every other instance. The
newest copy of each file always wins.

Works with Litematica (`.litematic`), WorldEdit (`.schem`, `.schematic`) and
structure NBT (`.nbt`) files out of the box.

## Settings

| Setting | Description |
|---------|-------------|
| Schematics folder name | Folder inside each instance game directory to sync (default `schematics`). |
| Show a notification after syncing | Pop up a message when files are copied. |
| Extra file extensions | Comma-separated extra extensions to treat as schematics. |

Open the settings from the **Extensions Settings** page: select this extension and
click the gear **Settings** button.
